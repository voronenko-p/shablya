## Configuring bot
Create a new [Slack bot](https://api.slack.com/apps/new) (see [tutorial](https://www.fullstackpython.com/blog/build-first-slack-bot-python.html))

```
export SLACK_BOT_TOKEN='your bot user access token here'
```

## Running the bot
```bash
python bot.py
#or
python3 bot.py
```
